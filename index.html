Runestone Interactive Project

The Runestone project aims to provide interactive, accessible and free eBooks to high-school students interested in learning CS. I worked on this project as a REU participant at the School of Interactive Computing. My responsibilities included contributing to the Runestone Components repository as a developer by finding bugs, improving UI/UX and making the eBooks more accessible to students with disabilities. I also write content for the StudentCSP eBook - a book for students preparing for College Board's AP CSP exam.

Halite Game Bot

Halite is an AI programming challenge in which players control a bot using a programming language of their choice to fight other bots on a 2D grid. As part of the Agency's Spring 2017 project, I wrote a heuristics-based bot in Python and helped in the deployment of a tournament structure made using Django to AWS. My bot did pretty well, I finished #3 on the leader-board.

SafeWalk

SafeWalk is an iOS application which uses crime logs available on GTPD's website and the Google Maps API to find the safest way to walk between any two points on campus. I hope to eventually expand the application for the entire city of Atlanta. Apart from making walking safer, the app can be used by foreign tourists to identify areas to avoid and can be used to make last-mile transit safer.
See GitHub repository.

DisMap

DisMap is an Android application which gives users the ability to avoid businesses with recorded incidents of racism, sexism, or otherwise intolerable behavior. It gives public places more transparency to allow users the right to be safe and/or avoid areas of conflict without having to figure it out the hard way.
Learn more about DisMap.
See Github repository.
