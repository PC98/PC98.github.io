Sample Works

Tsquare-Gradebook App

Tsquare is a portal that classes at Tech use for releasing grades, announcements, homeworks etc. Loading Tsquare on a mobile browser is a bad experience - the UI is plain HTML and the data loads slowly. Therefore I decided to make an iOS app that allows me to quickly view grades posted on Tsquare in a nice, user-friendly UI. What makes this project challenging is the fact that I don't have access to Tsquare's API, so my app must scrape raw HTML using Swift. I also managed to integrate the authentication process with Georgia Tech's CAS (which on it's own is easy to implement) and Duo's two-factor auth (this is/was tricky). While the app isn't complete yet, it is a work in progress. You can find my code for it here.

Runestone Interactive Project

The Runestone project aims to provide interactive, accessible and free eBooks to high-school students interested in learning CS. I worked on this project as a REU participant at the School of Interactive Computing. My responsibilities included contributing to the Runestone Components repository as a developer by finding bugs, improving UI/UX and making the eBooks more accessible to students with disabilities. I also wrote content for the StudentCSP eBook - a book for students preparing for College Board's AP CSP exam.

Face2Emojis

For Georgia Tech's first ever Meme-a-thon (a meme inspired Hackathon), I coded a web app that uses OpenCV to detect and replace faces in a user uploaded image with random emojis. I used PHP on the backend, and Python Imaging Library to deal with the actual logic. You can find my code here.

SafeWalk

SafeWalk is an iOS application which uses crime logs available on GTPD's website and the Google Maps API to find the safest way to walk between any two points on campus. I hope to eventually expand the application for the entire city of Atlanta. Apart from making walking safer, the app can be used by foreign tourists to identify areas to avoid and can be used to make last-mile transit safer.
See GitHub repository.

DisMap

DisMap is an Android application which gives users the ability to avoid businesses with recorded incidents of racism, sexism, or otherwise intolerable behavior. It gives public places more transparency to allow users the right to be safe and/or avoid areas of conflict without having to figure it out the hard way.
Learn more about DisMap.
